Написать регулярки для каждого вида обьектов
Разбить обьекты по видам. Один вид - одно обозначение. Каждый обьект имеет свой код. В итоге каждый обьект - "обозначение""код".

Виды:
    Идентификатор    -   названия разных обьектов
    Константы
        числовая
        символьная
    Операции        -    + - * / ... 
    Служебные слова 
    Разделители     -   ; , . []

Есть код на ПАскале. Т.е. строка.
Надо написать метод, который будет идти по этой строке, разбивать на лексемы. Возвращается строку, в которой каждая лексема заменена обозначением-кодом.


Метод, который проверяет лексему на принадлежность определнному виду. Возвращает какой-нибудь символ - обозначение вида.

Для каждого вида свой метод с регуляркой - это предикаты
